# Yassin Ahmed Portfolio

Bilingual English/Arabic portfolio for a PHP Laravel developer.

## Add your image
Replace `assets/profile.jpg` with your own photo using the same filename. Recommended size: 900×1100 px.

## Update your links
Edit `index.html` and replace:
- `YOUR_USERNAME`
- `YOUR_EMAIL@example.com`
- `YOUR_PHONE`

## Publish on GitHub Pages
1. Create a repository named `yassin-ahmed-portfolio`.
2. Upload all project files.
3. Open **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose `main` and `/root`, then save.
